# 📖 About the Project

The **Pravidya Admissions Platform** is a modern React-based frontend application developed to simplify and streamline the student admission process. It provides dedicated portals for administrators and counselors, enabling efficient lead management, admissions tracking, analytics, and secure role-based access through an intuitive and responsive user interface.

---

# ✨ Key Features

* 📝 Online Student Admission Form
* 👨‍💼 Admin Dashboard
* 👩‍🏫 Counselor Dashboard
* 🔐 Role-Based Authentication
* 🛡️ Protected Routes
* 📊 Lead & Admission Management
* 📈 Analytics Dashboard
* ⚡ REST API Integration
* 📱 Responsive User Interface

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/Pravidya-Frontend.git
```

### Install Dependencies

```bash
npm install
```

### Configure Environment

```env
VITE_API_URL=http://localhost:8000/api
```

### Start Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
